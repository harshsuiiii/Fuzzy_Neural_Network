# Fuzzy_Neural_Network
Convolutional Neural Networks (CNNs) have
emerged as powerful tools for financial time series analysis and
algorithmic trading. While the pooling layer is a critical component for dimensionality reduction, traditional methods like Max
Pooling (MP) and Average Pooling (AP) may not adequately capture the nuanced patterns in financial data. This study conducts
a comprehensive evaluation of ten different pooling strategies
- including traditional, fuzzy-based, and novel attention-based
methods - for cryptocurrency price prediction. Our extended
framework introduces four novel pooling layers (AAP, MSFP,
TAP, HSFP) alongside established methods. Experimental results
on Bitcoin data reveal that Adaptive Attention Pooling (AAP)
achieves the highest performance with 84.36
